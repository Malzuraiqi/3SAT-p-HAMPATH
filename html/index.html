Slide 1 · The Big Picture

3SAT Reduces to HAMPATH

The reduction goal

Given any 3CNF formula φ, build a directed graph G with nodes s and t such that G has a Hamiltonian path from s → t if and only if φ is satisfiable. This proves HAMPATH is NP-complete.

HAMPATH

{ ⟨G, s, t⟩ : G has a directed path visiting every vertex exactly once from s to t }

3SAT

{ ⟨φ⟩ : φ is a satisfiable Boolean formula in 3CNF — a conjunction of 3-literal clauses }

Slide 2 · The Variable Diamond

Variable Gadget: The Diamond

The truth encoder

For each variable xᵢ, we build a diamond. Any Hamiltonian path must traverse it in exactly one direction: left-to-right (TRUE) or right-to-left (FALSE) — no mixing allowed.

Slide 3 · Clause Gadget & Wiring

Clause Gadget: The Detour

Enforcing satisfaction

Each clause Cⱼ is a single node the path must visit. Edges are wired so the detour to cⱼ is only possible when the diamond is moving in the direction that makes a literal in Cⱼ true.

Slide 4 · Worked Example

Satisfying Assignment → Hamiltonian Path

Concrete example

Take φ = (x₁ ∨ x₂) ∧ (¬x₁ ∨ x₂). The assignment x₁ = FALSE, x₂ = TRUE satisfies φ. Below you can see the resulting graph and — highlighted in green — the Hamiltonian path from s to t that witnesses satisfiability.

The formula

φ = (x₁ ∨ x₂) ∧ (¬x₁ ∨ x₂)

C₁ = (x₁ ∨ x₂)  ·  C₂ = (¬x₁ ∨ x₂)

Satisfying assignment

x₁ = FALSE → row 1 goes RIGHT → LEFT
x₂ = TRUE  → row 2 goes LEFT → RIGHT

C₁ satisfied by x₂=T  ·  C₂ satisfied by x₂=T

Graph G for φ = (x₁ ∨ x₂) ∧ (¬x₁ ∨ x₂)  —  green path = Hamiltonian path (x₁=F, x₂=T)
Path direction: s → row 1 right-to-left (x₁=FALSE) → connector → row 2 left-to-right with detour to c₁ → continue → detour to c₂ → right end → t
Reading off the correspondence

Row 1 goes right-to-left → x₁ = FALSE.   Row 2 goes left-to-right → x₂ = TRUE.   The detours through c₁ and c₂ are both taken via row 2's LTR pass (x₂=TRUE makes both clauses true). Every node is visited exactly once. ✓

Slide 5 · Conclusion

Putting It Together

Satisfiable ⟹ Hamiltonian Path

For each variable, set the diamond direction to match its truth value. For each clause, pick any true literal and take the detour to cⱼ. Every node is visited exactly once.

Hamiltonian Path ⟹ Satisfiable

By normality, each diamond is traversed in a single direction. Read the directions as a truth assignment. Since every clause node was visited via a matching detour, every clause is satisfied.

φ is satisfiable  ⟺  G has a Hamiltonian path from s to t.  ∴ HAMPATH is NP-complete. □
Size of the reduction

Each diamond has 3k+1 internal + 2 top/bottom nodes = O(k). With n variables and k clauses: O(n·k) total nodes and edges — polynomial in |φ|. ✓
